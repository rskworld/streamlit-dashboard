## Quick Start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the dashboard:
   ```bash
   streamlit run app.py
   ```

   Or use the provided scripts:
   - Windows: `run.bat`
   - Linux/Mac: `./run.sh`
